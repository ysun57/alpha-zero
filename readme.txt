This project contains an Ultimate tic tac toe game board, a random player, a Q learning AI, and an alpha zero AI.
They are trained seperately.

Q learning training is outside run TrainQlearning.py to train a Q learner
AlphaZero training is inside. Run src.AlphaZero

The interface is outside. 
RunplayWithAlphaZeroPlayer1.py if you want to play first.
RunplayWithAlphaZeroPlayer2.py if you want to play second.
